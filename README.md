# Simple Scout - Football Wonderkid Database

Aplikasi web sederhana untuk memanajemen data pemain sepak bola muda (Wonderkids). Dibuat untuk memenuhi Tugas ke-1 Pra-Magang Reguler PT Makerindo (Web Programming).

## Fitur Utama (CRUD)
1. **Create**: Menambahkan data pemain baru beserta foto.
2. **Read**: Menampilkan daftar pemain dalam tabel interaktif.
3. **Update**: Mengedit informasi pemain (klub, potensi, posisi).
4. **Delete**: Menghapus data pemain dari database.

## Teknologi
- **Backend**: PHP Native
- **Database**: MySQL
- **Frontend**: Bootstrap 5 (CSS Framework)

## Cara Instalasi
1. Clone repository ini ke folder `www` (Laragon).
2. Buat database baru di PHPMyAdmin bernama `db_simplescout`.
3. Import file database atau jalankan query SQL pembuatan tabel.
4. Pastikan folder `uploads` tersedia untuk menyimpan gambar.
5. Buka browser dan akses `localhost/simple-scout-native`.
